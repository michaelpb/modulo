Demos

This is a collection of full-page Demo Apps. The current selection show more advanced capabilities.

  • FetchState API

    Demonstration of an API-backed, paginated book search app, without use of any JavaScript, beyond using the custom CPart FetchState.

  • JSX Templating

    Demonstration of integrating Babel JS and replacing Template CPart with custom CPart to enable React-style JSX-syntax in Modulo.
