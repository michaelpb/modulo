Take the tutorial!

Modulo is brand new in general, so it's probably brand new to you as well! The Modulo tutorial walks through the basics of setting up Modulo in a project and creating components.
Prerequisites: HTML, CSS (for part 1 and 2), and JavaScript (for part 3)

Starting snippet

Want to get started as quickly as possible? Copy & paste the following starting code into your HTML document:

Note: This is meant for exploring features. Your work will not be saved.

 
<!-- Include JS source, and define an embedded Component -->
<script Modulo src="https://modulojs.org/js/Modulo.js">
    <Component name="ButtonExample">
        <Props
            txt
        ></Props>
        <Template>
            <button><em>{{ props.txt }}</em></button>
        </Template>
        <Style>
            button { background-color: tomato }
        </Style>
    </Component>
</script>
