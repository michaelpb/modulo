Starting snippet (with CDN)

Want to get started as quickly as possible? Copy & paste the following starting code:

Note: This is meant for exploring features. Your work will not be saved.

 
<!-- Example custom component, using all core CParts -->
<template modulo-embed>
  <Component name="HelloWorld">
    <Props txt></Props>
    <Template> <a @click:=script.inc>{{ props.txt }} {{ state.num }}</a> </Template>
    <State num:=1 ></State>
    <Script> function inc() { state.num++; } </Script>
    <Style> a { color: green; } </Style>
  </Component>
</template>
