Example Library Demos

How to "fork" a Modulo.js demo

The tiles below link to a collection of full-page demonstration apps. These are intended as a reference of copy & paste recipes so you can get started even quicker with building Modulo apps.

Every demo can be "forked" (copied) by clicking the "COPY SOURCE" button in the top-right. Then, it can be pasted into a local HTML file in your text editor. The demos have been designed so they should "just work" if pasted in any HTML code.
